Variable Variables are containers for storing data values. We write variable in javascript with 'var'. In the right column sample we can see x, y, and z are variables <script>
var x = 11;
var y = 12;
var z = x + y;
document.getElementById("demo").innerHTML = z;
//if we have id 'demo' as DOM in html then it will give result 23 (from 11+12).
//To call id 'demo' we can put <p id="demo"></p> in our HTML body. </script>

Array JavaScript arrays are used to store multiple values in a single variable. [0] is the first element in an array. [1] is the second. Array indexes start with 0. Never put a comma after the last element. The effect is inconsistent across browsers. In the sample column we will see a variable 'cars' with single and multi dimensional arrays //single arrays
var cars = ["Saab", "Volvo", "BMW"];
//Multi dimensional arrays
var numeric = [ ['input1','input2'], ['input3','input4'] ];
numeric[0][0] == 'input1';
numeric[0][1] == 'input2';
numeric[1][0] == 'input3';
numeric[1][1] == 'input4';
; Semi colon. Always use semi colon after javascript statement. Even browser is smart enough to recognize our script but to standarize our script is to use semi colon after the statement document.getElementById("demo").innerHTML = "Hello Sun!";
Function Function is a block of code that we will use/render later. It can take arguments or value as well. We use function because we can reuse code: Define the code once, and use it many times. We can use the same code many times with different arguments, to produce different results. On the sample we will use the function to change meter to feet //we can put this script inside the html body to converse meter to feet. Variable 'meter' is able to change the meter value into feet
<p id="demoFoot"></p>
<script>
var meter = 60; // the value on meter is 60, but we can change it as much as we like
document.getElementById("demoFoot").innerHTML =
"The length is " + toFeet(3.28084) + " Feet";

Logic The control flow of your program. In JavaScript, logical operators are used for boolean logic where a boolean value may be returned depending on the outcome of an expression. Inside sample column on our right side is the sample of logic and how to test it <p>Assign 5 to x, and display the value of the comparison (x == 8):</p>

<script>
